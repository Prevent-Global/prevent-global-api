# Prevent.Global Mobile App Api

It is based on symfony framework.

## Deployment to Zenbox server

```bash
# SSH to Zenbox
cd /home/mash/domains/prevent.mash.stronazen.pl
git pull
composer install
cp -R public/. public_html/
```
